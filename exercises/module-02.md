These exercises support the vectors and geometry module.

## Concept Questions

1. Explain the difference between vector magnitude and vector direction.
2. What does a positive dot product tell you about two vectors?
3. Why is cosine similarity useful when comparing embeddings?
4. What is the difference between scalar projection and vector projection?

## Calculation Questions

1. Compute $\lVert [5,12]^{\mathsf T} \rVert_2$.
2. Compute $[2,-1,3]^{\mathsf T} + [0,4,5]^{\mathsf T}$.
3. Compute $[3,1]^{\mathsf T}\cdot[2,5]^{\mathsf T}$.
4. Find the distance between $[1,1]^{\mathsf T}$ and $[4,5]^{\mathsf T}$.
5. Find the angle between $[1,0]^{\mathsf T}$ and $[1,\sqrt{3}]^{\mathsf T}$.
6. Project $[6,2]^{\mathsf T}$ onto $[1,0]^{\mathsf T}$.
7. Compute the cosine similarity between $[1,2]^{\mathsf T}$ and $[2,4]^{\mathsf T}$.

## Coding Questions

1. Write a NumPy function that returns the dot product of two vectors.
2. Write a NumPy function that returns the angle between two non-zero vectors in degrees.
3. Write a NumPy function that projects one vector onto another non-zero vector.

## Short Application Questions

1. In a graphics lighting model, what does it mean if a surface normal has zero dot product with the light direction?
2. In a similarity search system, why might cosine similarity be preferred over Euclidean distance?
