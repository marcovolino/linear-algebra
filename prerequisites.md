This course starts from the basics, but students will move faster if they are comfortable with:

- algebraic manipulation;
- rearranging equations;
- coordinate geometry in two dimensions;
- functions and graphs;
- basic calculator or Python arithmetic.

Programming is optional at the start. Python and NumPy examples are included to connect the mathematics to engineering and AI practice.

## Self Check

Before starting, you should be able to:

1. Substitute values into a formula.
2. Expand and simplify simple algebraic expressions.
3. Read a point such as $(4,3)$ on a coordinate grid.
4. Interpret a table of numbers as measured data.

If these feel rusty, begin with the introductory lesson and spend extra time on the worked examples.
